Mes début en css

Link sont fichier html à sont fichier css

Tout d'abord nous devons faire notre base html puis utiliser la commande link pour lier notre fichier html à notre fichier css qu'on auras besoin a plus tard

<link rel="stylesheet" href="l'endroit de votre fichier style.css">

Une fois cela fait notre code donnera cela :

<!DOCTYPE html>
<html lang="en" dir="ltr">
<head>
 <meta charset="utf-8">
<link rel="stylesheet" href="assets/css/style.css">
<title></title>
 </head>
 <body>

 </body>
 </html>

Mettre un mot dans une phrase en italic

vous prenez le mot que vous voulez et vous le mettez entre les balises <span> puis vous devrez aller dans votre dossier style mettre span{font-style: italic} .
Important, si vous êtes sur Atom n'oublié pas de sauvegarder pour que le rendu ce face, pour sauvegarde rapidement CTRL+S

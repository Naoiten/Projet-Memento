Mes début en css

Link sont fichier html à sont fichier css

Tout d'abord nous devons faire notre base html puis utiliser la commande link pour lier notre fichier html à notre fichier css qu'on auras besoin a plus tard

<link rel="stylesheet" href="l'endroit de votre fichier style.css">

Une fois cela fait notre code donnera cela :

<!DOCTYPE html>
<html lang="en" dir="ltr">
<head>
 <meta charset="utf-8">
<link rel="stylesheet" href="assets/css/style.css">
<title></title>
 </head>
 <body>

 </body>
 </html>

Mettre un mot dans une phrase en italic

vous prenez le mot que vous voulez et vous le mettez entre les balises <span> puis vous devrez aller dans votre dossier style mettre span{font-style: italic} .
Important, si vous êtes sur Atom n'oublié pas de sauvegarder pour que le rendu ce face, pour sauvegarde rapidement CTRL+S

<!DOCTYPE html>

 <meta charset="utf-8">

Mettre des couleurs différente dans sont textes

Pour cela nous allons r'utilisez les balise span sauf que cette fois-ci nous alons utliser des numéros pour les différenciers. Nous voulons que deux mot sois en vert et deux mot en rouge.Nous allons utiliser pour l'exemple span1 et span2.

Le html donnera ça :

<!DOCTYPE html>
<html lang="en" dir="ltr">
<head>
<meta charset="utf-8">
<title></title>
<link rel="stylesheet" href="style.css">
</head>
<body>
<h4>Citizen Kane</h4>
<p><span1>kane</span1>, favori pour le poste de gourveneur, tient son discours de campagne. Au début des années 1940,<span1> Charles Foster Kane </span1> meurt dans son manoir de <span2>Xanadu</span2>, un vaste domaine palatial de <span2>Floride</span2>,en prononçant dans un dernier soufle <span>rosebud</span></p>
</body>
</html>

Le css donnera ça :

span{font-style: itali}
span1{color: red}
span2{color: green}

Une fois le html et le css lier cela nous donnera ça :

Citizen Kane

Kane, favori pour le poste de gourveneur, tient son discours de campagne. Au début des années 1940, Charles Foster Kane meurt dans son manoir de Xanadu, un vaste domaine palatial de Floride,en prononçant dans un dernier soufle rosebud.

changer la police

Pour changer la police rien de compliqué nous allons utliserons directement<p> pour prendre tout le paragraphe, comme ça tout les paragraphe seront dans la police que nous voulons, puis dans le css nous utiliserons la commande font-family puis mettre la police que nous voulons

Le html donnera ça :

<!DOCTYPE html>

<html lang="en" dir="ltr">
<head>
<meta charset="utf-8">
<title>Changer de police</title>
<link rel="stylesheet" href=style.css>
</head>
<body>
<h1>Test changer la police</h1>
<p> Test passer la police en Arial. </p>
</body>
</html>

Le css donnera ça :

p{font-family: Arial}

Une fois le html et le css lier cela nous donnera ça :

Test changer la police

Test passer la police en Arial.
